# EcoSphere – ESG Management Platform

EcoSphere is an enterprise-grade Environmental, Social, and Governance (ESG) Management Platform built to help modern enterprises track, analyze, report, and gamify their sustainability metrics.

## Tech Stack

### Frontend
- **Framework:** React with TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS & shadcn/ui
- **Navigation:** React Router
- **State Management & Data Fetching:** React Query & Axios
- **Charts:** Recharts

### Backend
- **Runtime:** Node.js (TypeScript)
- **Framework:** Express.js

### Database & ORM
- **Database:** PostgreSQL
- **ORM:** Prisma ORM

### Security & Utilities
- **Authentication:** JWT (JSON Web Tokens)
- **Validation:** Zod
- **File Upload:** Multer
